## 1. 0부터 5까지 반복

```swift
for i in 0...5 {
    print("호호 i: \(i)")
}
```

>   swift 에서 string이랑 같이 인자 출력할땐 <br>
>   print("string \(인자)") 이런식으로 표현하는 것 같음


## 2. 5보다 작은 수까지 반복

```swift
for i in 0..<5 {
    print("호호 i: \(i)")
}
```

>   i가 처음에는 0부터 시작 <br>
>   반복을 뜻하는 iterate 때문에 i로 쓰는게 일반적임


## 3. 0부터 4까지 짝수인 놈들만

```swift
for i in 0..<5 where i % 2 == 0 {
    print("호호 i: \(i)")
}
```
